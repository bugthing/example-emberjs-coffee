Example
=======

Skel to learn the structure and use of popular JS things.

Grunt - CoffeeScript - EmberJS - QUnit
--------------------------------------

Install all the stuff

    npm install

Start (build, test and watch for file changes)

    grunt

Connection to:

    http://localhost:9001/
